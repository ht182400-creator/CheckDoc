## Windows Terminal 配置 git 默认分支

新装的 WSL2 里 git push 后没显示，warning：`fatal: --local required`。

**解决方法：**

```bash
git config --global init.defaultBranch main
```

不要用 `master`，GitHub 已改为 `main` 默认。

## Vite HMR 缓存导致页面空白

修改 main.tsx 后页面 `#root` 完全为空，无 JS 错误。

**规避：**

```bash
Ctrl+C 停止 Vite
重新启动 start-frontend.bat
```

WSL 路径问题会导致 dev server 找不到模块。

## PowerShell UTF-8 编码

脚本输出中文乱码，`Out-File` 默认 ASCII。

**规避：** 在脚本顶部添加：

```powershell
$OutputEncoding = [System.Text.Encoding]::UTF8
chcp 65001 | Out-Null
```
