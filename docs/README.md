## 仓库结构

```
checkdoc/
├── docs/                 # 项目管理文档（本目录）
├── src/                  # 源码（见 02 架构设计）
│   ├── config.py         # 全局配置与字段 Schema（可扩展核心）
│   ├── logger.py         # 日志（毫秒级 + 每日轮转）
│   ├── scanner.py        # 目录扫描，发现 memory 目录与 MD 文件
│   ├── extractor.py      # 抽取引擎（规则 + LLM 可插拔）
│   ├── llm_client.py     # LLM 适配（OpenAI 兼容，可选）
│   ├── store.py          # 缓存/索引（增量扫描加速）
│   ├── ui_app.py         # NiceGUI 界面
│   └── main.py           # 入口
├── requirements.txt
└── README.md
```

## 关键链接

- 运行入口：`python -m src.main`
- 默认扫描根：`D:\Work_Area\AI`（可在 UI 中修改）
- 匹配规则：`**/memory/*.md`（递归查找名为 `memory` 的目录）
