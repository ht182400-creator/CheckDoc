## 日志规范

- 测试运行日志：`tests/test_run.log`（每次运行追加）
- 日志格式：`[时间.毫秒] PASS/FAIL/ERROR test_id`
- 失败案例含完整 traceback
- JSON Runner 使用 `_LoggedRunner` 类自动记录

## 新增用例流程

1. 编辑 `tests/test_data.json`，在对应模块数组中追加一行
2. 编辑本文件 (§09)，追加用例表格行
3. 运行 `python -m unittest tests.test_runner`
4. 检查 `tests/test_run.log` 确认 PASS
